## Setup & Development

### Prerequisites
- Node.js (v14 or newer)
- npm or yarn

### Installation
1. Clone the repository
2. Install dependencies:
   ```
   npm install
   ```
3. Start the development server:
   ```
   node app.js
   ```
4. Open your browser at `http://localhost:80`
